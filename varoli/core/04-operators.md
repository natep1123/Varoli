# 04 — Operators

Quick reference. For full definition see `./05-phonemic-schemas.md`.

Operators are not elements or forces. They do not belong to the same class as vowels or consonants. An operator acts on structure itself.

The operator y introduces controlled superposition into the system. It marks a position as unresolved, requiring that its value be determined through structural context rather than explicit assignment.

This allows the system to represent transition, superposition, and collapse without breaking compositional integrity. Meaning is not removed. It is deferred and then resolved.

The operator therefore does not add content. It governs how content becomes defined.

---

## y — Superposition-Collapse Operator

**Function:** Context-resolved collapse operator. Places a position into superposition: a state where what occupies that position is unresolved and must collapse from structural context. Not an element. Not a consonant. The mechanism by which unresolved states become resolved.

**Behavior:**
- In vowel position: marks an unresolved element awaiting context-driven collapse
- In consonant position: marks the active event of collapse within a structure
- At palindrome center: marks the attractor state as the collapse event itself

**Poetic:** The held breath before resolution. The white instant between states.
