# 06 — Generation Rules

This document defines how valid forms are constructed in Varoli. These rules govern structure, not interpretation.

---

## Rule 1: Valid forms are composed only of primitives

All valid forms are constructed exclusively from:

- Vowels (a, e, i, o, u)
- Consonants (attractor and general)
- Operator (y)

Any form containing elements outside this set is invalid.

---

## Rule 2: Structure must conform to valid patterns

Valid forms must follow one of the allowed structural patterns:

- CV (consonant → vowel)
- VC (vowel → consonant)
- Palindromic forms (e.g., aVa, eYe)

All valid structures must be reducible to primitive relationships.

---

## Rule 3: Directionality is structural

The order of primitives determines direction:

- CV: force applied to element (arrival)
- VC: element expressing through force (projection)

Reversing order changes function. CV and VC forms are not interchangeable.

---

## Rule 4: Palindromic closure defines stability

Palindromic forms represent structurally stable states.

- Attractor consonants center elemental palindromes
- Fire (e) centers on the operator: eYe

If a structure cannot resolve into a stable form, it is incomplete.

---

## Rule 5: The attractor state must be structurally derivable

Every valid form must resolve to an attractor state.

- The attractor is determined by structure, not assigned
- It must be structurally determinable

If the attractor cannot be determined, the form is invalid.

---

## Rule 6: Forms must reduce to primitives

All complex forms must be reducible to valid primitive relationships.

- No structure may introduce undefined behavior
- All components must resolve cleanly under decomposition

---

## Rule 7: The operator modifies but does not replace structure

The operator (y): 
- is not an element 
- is not a consonant 
- governs how unresolved positions collapse through structural context

Its presence must not violate structural rules.

---

## Summary

A valid form:

- uses only primitives
- follows structural patterns
- has a clear attractor state
- resolves under decomposition
- preserves directionality
