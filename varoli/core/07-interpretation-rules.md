# 07 — Interpretation Rules

This document defines how valid forms are interpreted. These rules govern meaning, not structure.

---

## Rule 1: Function is invariant

Each primitive has a fixed function.

- Function does not change with context
- Function defines all valid interpretations

---

## Rule 2: Interpretation cannot override function

Each primitive has a poetic field for human understanding.

- Poetic field is subordinate to function
- If interpretation conflicts with function, it is invalid

---

## Rule 3: Meaning is derived from structure

Meaning is not assigned externally.

- It is read from the arrangement of primitives
- The structure determines interpretation

---

## Rule 4: The attractor state defines the center of meaning

Every valid form resolves to an attractor state.

- Interpretation begins from the attractor
- The rest of the structure is read outward

---

## Rule 5: Scale realization does not alter function

A valid form has one invariant function.

That function may be realized at different scales without changing the underlying structure.

Scale includes:

- Internal vs external (self vs world)
- Temporal (momentary vs persistent vs eternal)
- Dimensional (surface vs depth vs breakthrough)

---

## Rule 6: Scale changes realization, not function

Changing scale does not create new meanings.

- It changes context only
- If interpretation requires altering function, it is invalid

---

## Rule 7: Interpretation follows structure, not expectation

A form must be interpreted based on its structure.

- Not based on prior language
- Not based on analogy
- Not based on assumption

---

## Rule 8: Multiple valid interpretations must resolve to the same function

A form may have more than one valid interpretation across different scales or contexts. All valid interpretations must be consistent with the same invariant function.

If two interpretations of the same form require different functions, at least one interpretation is invalid.

---

## Summary

A valid interpretation:

- preserves function
- respects structure
- resolves from the attractor
- remains consistent across scale realizations
