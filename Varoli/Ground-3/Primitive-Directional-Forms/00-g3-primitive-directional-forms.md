## 1. Purpose

This folder contains the primitive directional forms of Varoli.

A primitive directional form is the immediate two-character realization of primitive interaction:

- **CV** = kinetic arriving into substrate
- **VC** = substrate expressing through kinetic

These are not â€śverbsâ€ť in the inherited grammatical sense.
They are directional force-domain forms.

This folder exists to make the directional inventory:
- searchable
- learnable
- generative
- easy to validate

## 2. What is included

Each vowel file groups all primitive directional forms by **substrate**.

This means each file answers:

> What happens when every documented kinetic meets this one substrate?

Each file includes:
- the four attractor consonants
- the operator `y`
- the seventeen general consonants

That produces:
- 22 CV forms
- 22 VC forms

per substrate file.

Across five substrates, the full directional inventory is:
- **110 CV forms**
- **110 VC forms**
- **220 primitive directional forms total**

This matches the current primitive directional inventory. 

---

## 3. Organization rule

These files are organized by **vowel**, not by consonant.

This is intentional.

Grouping by vowel makes the system easier to:
- learn by domain
- use in scene-building
- use in conversational generation
- compare across one substrate at a time

---

## 4. Directionality rule

### CV
**Formula:** kinetic + substrate  
**Meaning:** force arriving into domain

### VC
**Formula:** substrate + kinetic  
**Meaning:** domain expressing through force

CV and VC are structurally distinct.
They are not stylistic reversals of one another. 

---

## 5. File structure

Each file uses the same compact format:

- consonant
- force label
- CV form
- compact CV gloss
- VC form
- compact VC gloss

This folder is a reference layer, not a full interpretive grammar.
Longer prose can be added later if needed, but the default should remain compact.

---

## 6. Classification rule

Everything in this folder is still **primitive directional form** inventory.

This folder does **not** cover:
- layered forms
- glottal-boundary forms
- stable worlds
- expanded stable worlds
- multi-unit chains

Those belong elsewhere.
