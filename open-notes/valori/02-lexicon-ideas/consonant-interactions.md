# Consonant Interactions (Exploratory)

## Observation

Single consonants represent individual forces.

Clusters represent:
→ interaction between forces

---

## CC Model

```

C1C2

```

→ C1 modifies C2

---

## CCC Model

```

C1C2C3

```

→ C1 and C3 modify C2

```

C1 → C2 ← C3

```

---

## Key Insight

Clusters are not new primitives.
They are compositional systems.

---

## Example

```

str

```

→ s and r both shaping t

---

## Constraint

All clusters must:
- reduce to individual components
- maintain directionality
- avoid symmetric ambiguity

---

## Risk

If clusters gain fixed meanings:
→ system collapses into memorization

---

## Open Questions

- How do larger clusters behave?
- Can clusters nest?
- Does order always remain directional?
