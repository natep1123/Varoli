# Verb Layering (Exploratory)

## Observation

Verbs in Varoli are not actions. They are modes of existence.

Base form:
```

CV

```

Example:
- va → relational projection
- ve → individuation
- vo → formation

---

## Layering Potential

Verbs can expand through:

1. Consonant interaction
```

CCV

```

2. Terminal constraints
```

CV + C

```

3. Tense/aspect suffixes
```

CV + suffix

```

---

## Emerging Pattern

Verb = core mode + constraints

---

## Example

```

va-m

```

→ ongoing relational mode

```

vo-k

```

→ completed formation

---

## Key Insight

Verb complexity scales structurally, not lexically.

No new verbs are required.
Only deeper compositions.

---

## Constraint

All layered verbs must:
- reduce to primitive components
- maintain invariant function
